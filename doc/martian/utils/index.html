Uncategorized collection of tiny useful functions

All the functions are simple wrappers around functions from other crates.

Functions

  • current_executable
    Parse the env::args() and return the name of the current executable as a String
  • make_path
    Given a path, file name, and extension, produce a file name with that extension.
  • obj_encode
    Shortcut function to encode an object as a Json dictionary
  • set_extension
    Given a filename and an extension, return the filename with the correct extension.
  • to_camel_case
    Convert the input to CamelCase
  • to_shouty_snake_case
    Convert the input to SHOUTY_SNAKE_CASE
  • to_snake_case
    Convert the input to snake_case
  • to_stage_key
    Given a path to the struct which may be fully qualified, return the struct name in lower snake case.
