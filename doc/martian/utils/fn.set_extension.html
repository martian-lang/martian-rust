Given a filename and an extension, return the filename with the correct extension.

Let’s say we have a file foo.a1.a2.a3. The extension() function associated with Path in rust returns a3 at the extension. This also means that p.set_extension("a2.a3") would be foo.a1.a2.a2.a3, which is probably not intended. This helper function accounts for paths with multiple dot extensions and sets up the filename correctly.

use martian::utils::set_extension;
assert_eq!(
    set_extension("/path/to/foo.bar.baz", "bar.baz.tmp"),
    std::path::PathBuf::from("/path/to/foo.bar.baz.tmp")
);
