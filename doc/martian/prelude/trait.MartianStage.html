A stage in martian which has split, main and join

For a toy example, see https://martian-lang.github.io/martian-rust/#/content/quick_start_split

In-process stage runner, useful for writing unit tests that exercise one of more stages purely from Rust. Executes stage with arguments args in directory run_directory. The defaul implementation executes split to get the stage definition (chunks), executes each chunk one after another and finally calls the join function.
