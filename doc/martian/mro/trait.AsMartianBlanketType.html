Expand description

A trait that defines how to convert this Rust type into an MartianBlanketType. Not all rust types can be converted to an MartianBlanketType. Not defined for

  • Unit, the type of () in Rust.
  • Unit Struct For example struct Unit or PhantomData<T>. It represents a named value containing no data. Any type which implements AsMartianPrimaryType also implements AsMartianBlanketType It is strongly recommended not to extend any types with this trait, instead use the AsMartianPrimaryType trait.
